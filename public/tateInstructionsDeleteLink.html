Tate Cartography Instructions

Deleting a link

Enter the names of the source and destination nodes that are connected by the link you'd like to delete. There are auto-complete options on all the fields. Select the name of the link between the nodes (as there might be more than one), once again there is an auto-complete option to assist you. On clicking the Delete button then you'll receive an indication if all went well.

The Modify Graph button will take you back to the Modify options whilst the Home button will let you see all the graphs available.
