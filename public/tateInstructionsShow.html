Tate Cartography Instructions

Managing Graphs

When the page first loads you'll see a list of graphs towards the top. These are divided into two sections - graphs that have been created by the project members - Tate Graphs, and graphs that you have created - Your Graphs.
Each graph is denoted by a name, an id and an author (if the graph has not been created by you). The image below illustrates.

Copying a graph

Next to each graph are a number of buttons that allow you to perform various actions on these graphs. If the graph does not belong to you then there is a copy option - this will copy the graph to your account, thereby allowing you to manipulate it. You cannot modify graphs that do not belong to you, although you can look at them. Each graph has a link to its page on the Graph Commons website by clicking on the link below the Graph ID column. It is often very useful to have the graph open in another tab on your browser whilst you perform the various actions on the graph.

Creating a new graph

If you'd like to generate a completely new graph then you'll need to supply a name for the graph and an optional description.
Once you've entered these then just click on New Graph and a new graph will be created in the Graph Commons account with the graph ID indicated.
When you create a new graph it will automatically be constructed with you as the author (by using your supplied user name).

This just creates an empty graph, we need to add some content. At this point you could just go the Graph Commons account and start adding items to the newly-created graph, which is absolutely fine. Alternatively if you have a lot of existing data that you'd like to add to the graph then a better way is to use a spreadsheet file.
From your spreadsheet file, save the file as a csv file (this stands for comma separated values - as it says all the data just separated with commas, very straightforward). If you then select the Choose File option and locate your csv file, load it and then hit the Generate graph button, the app will then create all the necessary nodes and links that form your graph.
If you need a sample file to test, you can download one here.
An indication of how this is progressing is given towards the bottom of the page, denoting how many nodes and links are going to be generated and the current progress.

When this is all done the page will refresh and you should see your newly created graph in the list of Your Graphs.

Modifying a graph

If you select the Modify option for your graph then you'll be presented with a range of options to update your graph. Instructions for completing these actions are given on the Modify Graph page.

Deleting a graph

Another option that you may perform on Your Graphs is to delete them. If you click the Delete button you will receive a confirmation message asking if you are sure that you really want to do this (you can't get it back once it's gone!). If you click OK then this graph will be deleted and the page will refresh to update the graphs in your collection.
