Tate Cartography Instructions

Adding a node

If you want to add a new node to your graph then you'll need to supply a name and a type. You also need a graph ID but this has been filled in for you.
You can enter any node name that you like. The Node Type has an auto-complete option if you want to add a node of an existing type, but you can also supply a new node type. Click the Add button to add the node, should this be successful then you'll see an appropriate indication.

The Modify Graph button will take you back to the Modify options whilst the Home button will let you see all the graphs available.
